# My Dev Portfolio

# This is my developer portfolio, a virtual business card where I share my work, skills and my story as a developer.

![portfolio-desktop](https://github.com/ReisLuizito/portfolio-luiz-guilherme/assets/131827303/1704efa6-5438-47a7-afd7-6255964d023e)

## About the project

The purpose of this project is to present my trajectory as a developer in a clear and attractive way, highlighting my most relevant skills and projects. The portfolio layout was developed using HTML and CSS, with Flexbox and Grid features to create a responsive and visually pleasing design.

## Resources Used

- HTML5
- CSS3
- Flexbox
- Grid

## How to use

1. Download or clone this repository.
2. Open the `index.html` file in your browser.
3. Browse the different sections of the portfolio to learn about my history, skills and projects.
4. Click on the project links to see more details and view the work done.
5. Please contact me using the information provided in the portfolio.
